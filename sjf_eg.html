Shortest Job First (SJF) Algorithm

Description

The SJF (Shortest Job First) algorithm selects the process with the smallest burst time from the ready queue for execution. It is a non-preemptive scheduling algorithm.

Processes are scheduled based on their burst times, where the shortest job gets executed first. If two processes have the same burst time, FCFS (First Come First Serve) order is followed.

Problem

Process Arrival Time Burst Time
P1 0 3
P2 1 5
P3 2 2

Solution

Step-by-Step Execution

  1. Process Arrival:
    • At time 0, only Process P1 is available in the ready queue.
    • At time 1, Process P2 arrives.
    • At time 2, Process P3 arrives.
  2. Process Execution:
    • Process P1 starts execution at time 0 and runs until its burst time is completed (0 to 3).
    • Process P3 starts execution from time 3 and runs until its burst time is completed (3 to 5).
    • Process P2 starts execution from time 5 and runs until its burst time is completed (5 to 10).
  3. Waiting Time:
    • Waiting Time of P1 = 0 - 0 = 0
    • Waiting Time of P2 = 3 - 2 = 1
    • Waiting Time of P3 = 5 - 1 = 4
  4. Turnaround Time:
    • Turnaround Time of P1 = 0 + 3 = 3
    • Turnaround Time of P2 = 4 + 5 = 9
    • Turnaround Time of P3 = 1 + 2 = 3
  5. Completion Time:
    • Completion Time of P1 = 3
    • Completion Time of P3 = 5
    • Completion Time of P2 = 10
  6. Average Waiting Time: (0 + 1 + 4) / 3 = 5 / 3 ≈ 1.67 units
