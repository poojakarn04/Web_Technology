First Come First Serve (FCFS) Algorithm

Description

The FCFS (First Come First Serve) algorithm is the simplest scheduling algorithm among all the scheduling algorithms. In this algorithm, processes are executed in the order they arrive in the ready queue. The process that arrives first gets executed first.

FCFS is a non-preemptive scheduling algorithm, which means that once a process starts executing, it continues until it completes its CPU burst. Only after that, the next process in the queue gets a chance to execute.

Problem

Process Arrival Time Burst Time
P1 0 5
P2 1 3
P3 2 4

Solution

Step-by-Step Execution

  1. Process Arrival:
    • At time 0, only Process P1 is available in the ready queue.
    • At time 1, Process P2 arrives.
    • At time 2, Process P3 arrives.
  2. Process Execution:
    • Process P1 starts execution at time 0 and runs until its burst time is completed (0 to 5).
    • Process P2 starts execution from time 5 and runs until its burst time is completed (5 to 8).
    • Process P3 starts execution from time 8 and runs until its burst time is completed (8 to 12).
  3. Waiting Time:
    • Waiting Time of P1 = 0 - 0 = 0
    • Waiting Time of P2 = 5 - 1 = 4
    • Waiting Time of P3 = 8 - 2 = 6
  4. Turnaround Time:
    • Turnaround Time of P1 = 0 + 5 = 5
    • Turnaround Time of P2 = 4 + 3 = 7
    • Turnaround Time of P3 = 6 + 4 = 10
  5. Completion Time:
    • Completion Time of P1 = 5
    • Completion Time of P2 = 8
    • Completion Time of P3 = 12
  6. Average Waiting Time: (0 + 4 + 6) / 3 = 10 / 3 = 3.33 units
